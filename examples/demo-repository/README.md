# Lightfold Grid Demo Repository

This deliberately small repository is used by the five-minute onboarding workflow.

Suggested safe task:

> Add tests for repeated punctuation and update `slugify` so punctuation is removed,
> repeated separators collapse, and empty input returns an empty string. Run `npm test`.

The task is narrow enough to inspect manually while still exercising planning, coding,
testing, review, and Lightfold Grid message routing.
