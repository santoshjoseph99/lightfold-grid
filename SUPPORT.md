# Support

Lightfold Grid is experimental open-source software maintained on a best-effort basis.

## Community Support

Use GitHub issues for reproducible bugs and feature requests. Use repository discussions
for setup questions, design ideas, and general usage help when discussions are enabled.

Before requesting help:

- Read the README and known limitations.
- Search existing issues.
- Run the health checks and deterministic test suite.
- Remove secrets and proprietary content from logs or diagnostics.

## Security And Privacy

Do not post vulnerabilities, credentials, private prompts, proprietary repository
content, or unredacted diagnostic bundles publicly. Follow [SECURITY.md](./SECURITY.md)
for vulnerability reports.

## Boundaries

Maintainers do not guarantee:

- Support for every agent CLI, model, provider, operating system, or repository.
- Recovery from destructive commands or YOLO-mode behavior.
- Compatibility with undocumented provider or CLI changes.
- Individual debugging, response times, uptime, or production suitability.

Commercial support and service-level agreements are not currently offered.
